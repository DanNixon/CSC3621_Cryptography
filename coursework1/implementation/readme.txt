Notes on executing supplied code

The supplied code is provided as a Maven project inside the this directory, the
project can be built using:

mvn package

After the project has been built the included Bash scripts can be executed from
this directory.

Code for each exercise can be found in:
src/main/java/com/dan_nixon/csc3621/cw1/ex1 - Exercise 1
src/main/java/com/dan_nixon/csc3621/cw1/ex2 - Exercise 2
src/main/java/com/dan_nixon/csc3621/cw1/ex3 - Exercise 3

The code has been tested on OpenJDK 6 and 7 and Oracle JDK 7 and 8, the results
included in the reports were obtained using OpenJDK 7.
